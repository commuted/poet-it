Poetit
Copyright 2026 Ron Hinchley — MIT License, except where otherwise noted

A poetry editor with meter analysis, rhyme detection, and linguistic tools.

Features:
  - Line-by-line text editing with syllable counting
  - Metric stress analysis (prosodic library)
  - Rhyme scheme detection and lookup
  - Word definitions and dependency diagrams
  - Thesaurus integration
  - Spell check with inline suggestions
  - Version control (dulwich git compatible repo)

Libraries:
